## ⚙️ 단계별 밸런스 조정 전략

### Phase 1: 즉시 적용 가능 (값 조정만)
```
✅ 벌써 구현된 스킬 → trigger_chance, value, duration만 수정
예) 성기사 발도: 15% → 18% (Lv.6)
예) 충격로봇 전기충격: 1.125초 → 1.6875초 (Lv.12)
```

### Phase 2: 로직 추가 필요
```
⚠️ 현재 미구현 영웅 스킬 처리
예) nth_attack (n번째 공격마다)
예) 팀 버프 (passive_aura with team_attack_speed_bonus)
예) 조건부 피해 (스턴 상태 시 피해 증폭)
예) 지속 디버프 (10초동안 방어력 20 감소)
```

### Phase 3: 고급 메커닉 (필요시 호출)
```
🟡 게임 메커닉 확장 필요
예) 필드 오브젝트 (덩굴, 번개 구슬)
예) 거리 비례 피해
예) 소환 유닛 (분신, 임프)
```

## 🔧 Phase 1 패치: 즉시 적용 값 조정

### 적용 파일: `config/hero_data.php`

```php
// 1. 성기사 [발도] - Lv.6 확률 증가
'level_up' => array(
  6 => array(
    'description' => '발동확률 20% 증가',
    'effects' => array(
      array('type' => 'trigger_chance_increase', 'value' => 20)
    ),
  ),
  12 => array(
    'description' => '피해량 50% 증폭',
    'effects' => array(
      array('type' => 'damage_aoe_magic', 'value' => 675)  // 450 * 1.5
    ),
  ),
),

// 2. 충격로봇 [전기 충격] - Lv.12 기절 시간 50% 증폭
'level_up' => array(
  12 => array(
    'description' => '기절 시간 1.6875초로 증폭',
    'effects' => array(
      array('type' => 'stun', 'duration' => 1.6875)
    ),
  ),
),

// 3. 사냥꾼 [방어력 감소] - Lv.12 방어력 50% 증폭
'level_up' => array(
  12 => array(
    'description' => '방어력 감소 22.5로 증폭',
    'effects' => array(
      array('type' => 'shred_armor_flat_aura', 'value' => 22.5)
    ),
  ),
),

// 4. 보안관 [속사] - 상시 효과로 조정 (6초 한정 → 항상 활성)
// 20% 확률로 공격 속도 30% 증가 (턴제 적응)
'effects' => array(
  array('type' => 'extra_attack_chance', 'value' => 30),
),

// 5. 닌자 [암살] - trigger_chance 실제값 조정
'trigger_chance' => 16.5,  // 현재 5% → 16.5%로 수정

// 6. 닌자 [암살] - Lv.12 발동확률 보정
'level_up' => array(
  12 => array(
    'description' => '발동확률 65% 증폭',
    'effects' => array(
      array('type' => 'trigger_chance_boost', 'value' => 65)  // 16.5 * 1.65 ≈ 27%
    ),
  ),
),
```
